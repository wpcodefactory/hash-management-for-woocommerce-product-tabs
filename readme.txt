Offer a better experience for your users by letting the WooCommerce product tabs to interact with the URL hash.

== Description ==

**Hash Management for WooCommerce Product Tabs** does only two things:

* Adds a hash to URL when a WooCommerce product tab is clicked.
* Opens the correct WooCommerce product tab considering the hash from the URL.

This will provide a better experience for the users as they will be able to use the backward/forward browser buttons to navigate through the tabs and they will be able to access links that go from one tab to the other without the need of page reload.

== Installation ==

1. Upload the entire plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
